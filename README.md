# ⚡ ResumAI 

Um agregador e sumarizador automático de notícias do Brasil que utiliza Inteligência Artificial para gerar resumos direto ao ponto.

## 🚀 Funcionalidades

* **Web Scraping:** Coleta em tempo real das principais manchetes do G1.
* **Integração com API:** Busca notícias recentes através da NewsAPI.
* **Processamento de Linguagem Natural (NLP):** Sumarização automática em português utilizando o modelo de IA `PTT5`.
* **Pronto para Mobile:** Estruturado com uma API em FastAPI pronta para abastecer um aplicativo celular.

## 🛠️ Tecnologias Utilizadas

* **Python 3**
* **FastAPI** (Construção da API)
* **Hugging Face Transformers & PyTorch** (Modelo de IA)
* **BeautifulSoup4** (Web Scraping)
* **Uvicorn** (Servidor ASGI)
